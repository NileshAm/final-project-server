# Final project Backend
## Runs the app in the development mode.
`npm run dev`\
And open [http://localhost:5000/test](http://localhost:5000/test) to view it in your browser.

## Runs the app in the production mode.
`npm run start`\
And open [http://localhost:5000/test](http://localhost:5000/test) to view it in your browser.


## To exit either mode
Press `ctrl + c`\
then `y` and **Enter**.
